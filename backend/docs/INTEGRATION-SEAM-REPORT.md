## Mission Executor Dispatch

**Location**: `app/services/mission_executor.py`

**Pattern**: Hardcoded match/block (lines 451-467)

```python
match task.task_type:
    case "llm" | "llm_call":
        result = await self._execute_llm(task, input_data, mission)
    case "tool" | "tool_execution":
        result = await self._execute_tool(task, input_data, mission)
    case "rag" | "rag_query":
        result = await self._execute_rag(task, input_data)
    case "web_search":
        result = await self._execute_web_search(task, input_data, mission)
    case "code" | "code_execution":
        result = await self._execute_code(task, input_data, mission)
    case "file_operation":
        result = await self._execute_file(task, input_data, mission)
    case "review" | "human_review":
        result = await self._request_human_input(db, mission, task)
    case _:
        result = {"success": False, "error": f"Unknown task type: {task.task_type}"}
```
